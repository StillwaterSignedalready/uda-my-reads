## 功能(显示图片需翻墙)
该应用是一款书架应用，用于搜索用户感兴趣的书籍，并归类到不同的书架。点击右下角的加号切换至搜索功能，再点击左上角的箭头切换回书架。每本书右下角的dropDown键可切换书所在的书架。

## 技术细节
应用由React + react-router实现，用PUT和GET方法与服务器交互数据。
